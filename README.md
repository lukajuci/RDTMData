## Overview

This repository contains the device firmware for the RDTM Data node using the Meshtastic project as a base.

Changes include writing and implementing the [DataNodeModule](https://github.com/lukajuci/RDTMData/blob/main/src/modules/DataNodeModule.cpp ) as well as the [rdtmtelemetry](https://github.com/lukajuci/RDTMData/blob/main/protobufs/meshtastic/rdtmtelemetry.proto) protobuf for packaging the desired sensor data.

- **[Meshtastic Project Main Github](https://github.com/meshtastic)**
